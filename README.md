## 常见问题

**Q: 4WS_DUMMY是什么？**
A: 材料是否需要进入四穿库的标记，系统据此推荐入库口9或10。

**Q: 任务滞留超过30分钟怎么办？**
A: 登录WES系统检查原因，根据情况取消或强制完成。

**Q: 如何查询库存？**
A: 登录WES系统，选择"库存"菜单，输入托盘号查询。

**Q: 如何处理BCR扫码失败？**
A: 检查条码清晰度、清洁镜头，必要时手动输入。
