# 异常代码 / Error Codes

## 入库异常 (Inbound)

| 代码 | 描述 | 原因 | 处理方案 |
|------|------|------|----------|
| E101 | BCR扫码异常 | 条码不清晰/镜头脏 | 清洁镜头/手动输入 |
| E102 | 超高超宽 | 物料尺寸超限 | 转地面排位或人工 |
| E103 | 托盘倾斜 | 放置不平 | 人工整理后重试 |
| E104 | 物料超重 | 超托盘承重 | 转地面排位 |
| E105 | 物料超高 | 超层高限制 | 转BCP排位 |
| E106 | 类型不允许 | Item Family限制 | 转BCP排位 |
| E107 | 无库位 | 库位已满 | 转地面排位 |
| E108 | 输送带卡料 | 物料阻塞 | 复位/手动处理 |
| E109 | 重复扫码 | 托盘号重复 | 人工确认跳过 |
| E110 | 卡托盘 | 托盘变形 | 复位/强制失败/手动 |
| E111 | 库位锁定 | 库位异常 | 解锁或换库位 |
| E112 | 任务丢失 | 系统异常 | 重新下单 |

## 出库异常 (Outbound)

| 代码 | 描述 | 原因 | 处理方案 |
|------|------|------|----------|
| E201 | 质量状态变更 | 出库时状态非U | 执行→move_error |
| E202 | 取消失败 | 任务执行中取消 | 继续执行→move_error |
| E203 | 无匹配库存 | 库存不足 | 检查Prime库存 |
| E204 | 出库超时 | 任务>30min | 强制完成/取消 |
| E205 | 托盘损坏 | 托盘异常 | 检查托盘状态 |
| E206 | 库存锁定 | 库位锁定中 | 解锁后重试 |
| E207 | 四穿取货失败 | 设备故障 | 检查四穿状态 |

## 倒库异常 (Move)

| 代码 | 描述 | 原因 | 处理方案 |
|------|------|------|----------|
| E301 | 目标库位占用 | 有其他托盘 | 换库位或先倒库 |
| E302 | 源库位为空 | 无物料 | 检查源库位 |
| E303 | 理货任务失败 | 设备异常 | 重新执行 |

## 任务状态 (Task Status)

| 状态 | 可取消 | 可改优先级 | 可强制完成 | 可强制失败 |
|------|--------|------------|------------|------------|
| 新建 / New | ✅ | ✅ | ✅ | ✅ |
| 受理 / Assigned | ❌ | ❌ | ✅ | ✅ |
| 执行 / Executing | ❌ | ❌ | ✅ | ✅ |
| 完成 / Completed | ❌ | ❌ | ❌ | ❌ |
| 失败 / Failed | ❌ | ❌ | ❌ | ❌ |

| E302 | IAC连接失败 | WebSocket断开 | 重连/重启服务 |
| E303 | WCS通信中断 | MQTT断开 | 检查网络/重启 |
| E304 | 数据库连接失败 | DB连接问题 | 检查数据库 |
| E305 | 数据库异常 | SQL错误 | 查看日志 |
| E306 | PLC通信失败 | TCP连接断开 | 检查网络 |
| E307 | 设备超时 | 设备无响应 | 检查设备状态 |
| E308 | 配置错误 | 系统配置异常 | 检查配置 |

## 指示灯状态

| 颜色 | 状态 | 说明 |
|------|------|------|
| 绿色 | 可放货/可取货 | 正常状态 |
| 红色 | 出库中/不可操作 | 等待完成 |
| 黄色 | 异常/需人工 | 检查处理 |
| 闪烁 | 任务执行中 | 正常工作中 |

## 报警级别

| 级别 | 说明 | 影响 |
|------|------|------|
| P1 | 紧急停产 | 全部停机 |
| P2 | 设备故障 | 部分停机 |
| P3 | 轻微异常 | 提醒处理 |
| P4 | 信息提示 | 日常提醒 |
