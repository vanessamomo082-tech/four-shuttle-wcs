# 异常代码与处理

## 入库异常

### 异常类型

| 异常码 | 描述 | 处理方案 |
|--------|------|----------|
| E101 | BCR扫码异常 | 输送线回退 → 异常口 |
| E102 | 超高超宽 | 输送线回退 → 异常口 |
| E103 | 业务异常 | 输送线回退 → 异常口 |
| E104 | 重复入库 | 回告Prime拒绝 |
| E105 | 库位已满 | 异常口 → 人工处理 |
| E106 | 物料类型不允许 | 转BCP排位，检查item family |
| E107 | 无可用库位 | 转入库旁地面排位 |
| E108 | 相同托盘号库存 | 转地面排位，调查差异 |
| E109 | 数量大于板规 | 转地面排位，维护footprint |

### 处理流程

1. WES下发输送线回退任务给WCS
2. WCS转发给PLC执行回退
3. 回退完成后，WES向IAC下发去异常口的搬运任务
4. 异常口大屏显示异常信息
5. 人工处理完成后，WES自动下发入库induction

## 出库异常

### 异常类型

| 异常码 | 描述 | 处理方案 |
|--------|------|----------|
| E201 | 质量状态变更 | 继续执行 → 回传move_error |
| E202 | 取消失败 | 继续执行 → 回传move_error |
| E203 | 托盘异常 | 异常口 → 人工处理 |
| E204 | 找不到库存 | 回告Prime无库存 |
| E205 | 没有匹配库存 | 检查Prime库存比对 |

### 屏幕显示

- 托盘号
- 异常原因："出库下架中变更质量状态" / "取消失败"

## 任务异常

### 状态处理

| 任务状态 | 可取消 | 可修改优先级 | 可强制完成 |
|----------|--------|-------------|------------|
| 新建 | ✅ | ✅ | ✅ |
| 受理 | ❌ | ❌ | ✅ |
| 执行 | ❌ | ❌ | ✅ |
| 完成 | ❌ | ❌ | ❌ |
| 失败 | ❌ | ❌ | ❌ |

### 优先级规则

- 数值越低，优先级越高
- 新建状态可修改优先级
- 受理/执行状态不可修改

## 虚拟容器异常

### 异常类型

| 异常码 | 描述 |
|--------|------|
| E401 | 库位组有任务，无法创建 |
| E402 | 库位组已被占用 |
| E403 | 虚拟托盘命名冲突 |

### 处理

- 检查库位组是否有受理/执行中的任务
- 如有，提示"该库位不可用"
- 如无，创建成功并锁闭库位组
